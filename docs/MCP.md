# MCP Usage

## Serena
Used for:
- Finding symbols
- Refactoring safely
- Understanding dependencies

Must be used before modifying unfamiliar code.

## Context7
Used for:
- Framework documentation
- API changes
- Version-specific behavior

## Brave Search
Used only if:
- Info not in repo
- Info not in Context7

## Playwright
Used for:
- E2E flows
- Behavior verification
- UI regression

## Security
- Never expose secrets
- Use .env
- Do not connect to production without approval
